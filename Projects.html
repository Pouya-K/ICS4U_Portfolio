Programming is a skill that has been gaining a lot of popularity in recent times. It has become an essential skill in various fields, including business, engineering, and technology. My journey with programming started in grade 9, where I was introduced to the language of processing. Processing is a language that is designed to make it easy to program interactive graphics and animations. I found the language fascinating and challenging at the same time. It required a lot of patience and practice, but I was determined to learn it. I spent many hours practicing and experimenting with the language, and eventually, I was able to build some exciting projects.

During the summer after grade 9, I decided to expand my programming knowledge by learning python. I turned to youtube tutorials as a resource to help me learn python. The tutorials provided me with a solid foundation in python, and I was able to build some small projects like a calculator and a simple game. I enjoyed the process of learning python, and it gave me a lot of confidence in my programming abilities.

For the past year, I have been learning Java, as well as HTML, Javascript, and CSS. Java is a powerful programming language that is used extensively in enterprise applications. I found Java to be challenging, but I was determined to learn it. HTML, Javascript, and CSS are the building blocks of the internet. Learning these languages was essential in understanding how web development works. I have been able to build some exciting web applications using these languages, and it has been a lot of fun. After building a solid foundation on HTML, I created a website with the help of my friend, Luka Jovanovic, designed to help grade 10 students with their math homework.
